fre:ac-Benutzerhandbuch

Eine Anleitung zum CD-Rippen für Anfänger und erfahrene Benutzer gleichermaßen

fre:ac ist ein vielseitiger CD-Ripper, Codierer und Konverter für Audiodateien. Es unterstützt unter anderem die Formate MP3, Ogg Vorbis und Wave. Sie können die neueste fre:ac-Version von der fre:ac-Homepage herunterladen.

Mit fre:ac können Sie qualitativ hochwertige MP3-Dateien erstellen, die Sie z.B. auf Ihrem PC oder Ihrem portablen Player abspielen können.

Das Benutzerhandbuch wird ständig weiterentwickelt. Die Anfängeranleitung ist fertig, aber an der Anleitung für erfahrene Benutzer gibt es noch ein wenig Arbeit zu tun.
